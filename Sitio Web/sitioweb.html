Windows 95 4.0 1995 Intel Pentium o compatible 66 MHz o más x86 1 SDRAM 4 MB o más 16-300 MHz (procesadores Intel 80386 y 80486) Plug and Play Introdujo el menú de inicio, la barra de tareas y nombres de archivos largos. Primera versión con Plug and Play.

SDRAM 64 MB o más 16-300 MHz (procesadores Intel Pentium II y Pentium III) Soporte para ACPI Orientado a negocios, mayor estabilidad y seguridad, soporte para sistemas de archivos NTFS.
Windows XP 45296 2001 Intel Pentium III o superior 233 MHz o más x86 Basado en Windows 2000, con soporte para un número similar de hilos por proceso. DDR SDRAM 128 MB o más 1.3 - 3.8 GHz (procesadores Pentium 4, Athlon XP y posteriores) Soporte para Wi-Fi Nueva interfaz gráfica, mejor gestión de redes y multimedia, más estabilidad y soporte para varias cuentas de usuario.
Windows Vista 6.0 2007 Intel Pentium 4 o superior 1 GHz o más x86/x64 Mejoras en el manejo de hilos y capacidad para manejar múltiples hilos de manera más eficiente. DDR2 SDRAM 512 MB o más 1.8 - 3.4 GHz (procesadores Intel Core y AMD Athlon 64 X2) Aero Glass Nueva interfaz Aero, mejores capacidades de búsqueda, seguridad mejorada con UAC.
Windows 7 45297 2009 Intel Core 2 o superior 1 GHz o más x86/x64 Continuó con el modelo de manejo de hilos de Windows Vista, con mejoras en la estabilidad y rendimiento. DDR3 SDRAM 1 GB (32-bit) o 2 GB (64-bit) o más 1.8 - 3.5 GHz (procesadores Intel Core 2 Duo, Core i3/i5/i7, AMD Phenom II) Soporte para multitouch Mejor rendimiento, interfaz más pulida, mejor soporte para dispositivos táctiles.
Windows 8 45328 2012 Intel Core i3 o superior 1 GHz o más x86/x64 Mejoras en el rendimiento general, incluyendo el manejo de hilos. Windows 8.1 45357 2013 Intel Core i3 o superior 1 GHz o más x86/x64 Mejoras adicionales en el rendimiento y gestión de hilos respecto a Windows 8. DDR3 SDRAM 1 GB (32-bit) o 2 GB (64-bit) o más 1.8 - 4.0 GHz (procesadores Intel Core i3/i5/i7, AMD FX) Mejoras en UEFI Mejoras en la interfaz de usuario, regreso del botón de inicio, mejor integración con servicios en la nube.
Windows 10 10.0 2015 Intel Core i3 o superior 1 GHz o más x86/x64 Soporta un alto número de hilos por proceso, similar a versiones anteriores, con mejoras en la eficiencia. DDR3/DDR4 SDRAM 1 GB (32-bit) o 2 GB (64-bit) o más 1.8 - 5.0 GHz (procesadores Intel Core i5/i7/i9, AMD Ryzen) Soporte para DirectX 12 Interfaz unificada para diferentes dispositivos, Cortana, navegador Edge, actualizaciones frecuentes.
Windows 11 10.0 2021 Intel Core i5 de 8ª generación o AMD Ryzen 5 2000 o superior 1 GHz o más x86/x64 Similar a Windows 10 en términos de manejo de hilos, con optimizaciones adicionales para el rendimiento y la gestión de tareas. DDR4 SDRAM 4 GB o más 1.8 - 5.0 GHz (procesadores Intel Core i5/i7/i9, AMD Ryzen) Soporte para TPM 2.0 Nueva interfaz de usuario, mejoras en rendimiento y seguridad, mejor integración con aplicaciones Android.
